## Design Context

### Users
Équipe interne Cocktail Média (2-4 personnes : Marie-Christine, Félix, etc.). Créateurs de contenu social qui produisent des visuels pour les réseaux sociaux de leurs clients sportifs. Contexte : bureau, desktop uniquement, usage quotidien rapide. Pas d'accès public — outil interne.

### Brand Personality
- **Cocktail Média** : agence de contenu sport · professionnel · bilingue FR · direct · efficace
- **Royal de Shawinigan (LNHBF)** : hockey féminin · rouge vif · bold · communautaire
- **Xtreme de Shawinigan (LNHB)** : hockey masculin · marine/or · élite · dynamique
- Trois mots : **bold · sporty · pragmatique**
- Émotions visées : efficacité, fierté d'équipe, impact visuel immédiat

### Aesthetic Direction
- Dark mode uniquement (contexte de travail, écrans multiples, usage prolongé)
- Polices display : Bebas Neue · Impact · condensed bold → sports authentique
- Palette : rouge royal (#d62127), marine Xtreme (#062045), or (#FFC702), orange CM (#e8640a)
- Pas de gradients texte, pas de glassmorphism, pas d'esthétique "AI générique"
- Templates 1080×1080 (Instagram square) · 1080×1920 (Story/Reel)
- Outil desktop uniquement — mobile non requis

### Design Principles
1. **Pragmatique avant tout** : chaque champ d'éditeur doit avoir un impact visible immédiat sur le template
2. **Branding sport cohérent** : les templates doivent ressembler à une production de chaîne sportive, pas à un outil SaaS générique
3. **Exportation fiable** : le PNG généré doit être pixel-parfait à 1080px (scale 3× pour retina)
4. **Modifiable sans limites** : couleurs, polices, contenus — tout doit être personnalisable via l'UI
5. **Vitesse d'exécution** : un post doit pouvoir être produit en moins de 2 minutes
