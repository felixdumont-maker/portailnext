# Social Graphics UI Kit — Cocktail Média · Royal de Shawinigan

Outil de génération de visuels sociaux 1080×1080 (Instagram) et 1080×1920 (Stories/Reels).

**Ouvrir `cocktail.html`** pour accéder à l'outil complet.

## Templates disponibles

| Série | Templates | Description |
| --- | --- | --- |
| **T** (Cocktail Média) | T01–T09 | Annonces, promos, témoignages, absence, etc. |
| **R** (Royal / Xtreme) | R01–R11 | Match, résultats, horaire, classement, joueur·euse |

## Architecture

Chaque template est un fichier JSX autonome avec ses `defaults`. L'éditeur est dans `cocktail.html` (4500+ lignes). Pas de build step — ouverture directe dans le navigateur.
