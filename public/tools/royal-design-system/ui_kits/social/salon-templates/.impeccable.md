## Design Context

### Users
Équipe interne Cocktail Média produisant les visuels sociaux pour **Les Belles Sœurs**, salon d'esthétique à Shawinigan, QC. Usage desktop, bureau, production rapide de posts Instagram/LinkedIn/Stories. Les templates sont générés en PNG via Playwright (Omelette).

### Brand Personality
- **Les Belles Sœurs** : salon d'esthétique · Shawinigan · ouvert depuis 2025
- Valeurs : Rituel · Savoir-faire · Authenticité
- Trois mots : **Intime · Éditorial · Bienveillant**
- Émotions visées : calme, confiance en soi, sentiment d'être prise en charge par de vraies expertes
- Phrase signature : "Sortez d'ici plus vous-même que jamais."

### Voix & ton
- Chaleureux sans être familier. Expert sans être condescendant.
- Sensoriel : décrire ce qu'on ressent, pas ce qu'on fait.
- Verbes doux : révéler, cultiver, retrouver, accompagner.
- Phrases courtes. Respirations. Pas de bavardage.
- Jamais d'argot ni de superlatifs vides ("incroyable", "wow").
- Ancrage local : Shawinigan, l'équipe, le rituel — pas du contenu Pinterest générique.

### Aesthetic Direction
- Palette : crème chaud (#FFFCFA) · blush (#F8CBCA) · pêche (#FDEAE3) · camel (#C8A07A) · encre brun foncé (#2C1F1C) · rose accent (#C98A87)
- Typographie : **Gloock** (éditorial serif, signature émotionnelle en italique) + **Figtree** (corps — utiliser 300 Light pour le body, 400 pour les labels, 500 max pour les CTA)
- Thème clair. Crème et blush dominent. L'encre foncée pour les versions premium.
- Templates 1080×1080 (feed) · 1080×1920 (story/reels)

### Design Principles
1. **Sensoriel avant informatif** : le copy doit évoquer une sensation, pas juste transmettre un fait
2. **Légèreté typographique** : Figtree 300 pour le body, jamais de fontWeight >500 sur les labels — la hiérarchie vient du contraste serif/sans, pas du gras
3. **Ancré localement** : chaque template doit pouvoir mentionner Shawinigan, l'équipe, un service réel du salon
4. **Éditorial, pas promotionnel** : préférer le ton magazine beauté raffiné à l'affichette de promo
5. **Rituel, pas rendez-vous** : la visite au salon est une expérience — le langage doit le refléter
